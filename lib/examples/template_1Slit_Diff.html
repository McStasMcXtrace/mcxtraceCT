The template_1Slit_Diff Instrument

An example instrument showing single slit diffraction

  • Origin: DTU Physics
  • Date: Jan 2014

Description

A template instrument that shows how single slit diffraction works.
The slit is positioned 1m downstream from a point source. The slit
width may be varied. The slit height is set to .8e-6 m.

Parameters in boldface are required; the others are optional.
