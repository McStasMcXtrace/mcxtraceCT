The Sandpaper Component

Identification

  • Author: Matteo Busi
  • Origin: DTU Physics
  • Date: March 2017
  • Version: 1.0

Description

Sample used to generate sandpaper speckle patterns. It is modeled as a number
of spheres centered in the same z-plane, with particles' size following 
normal distribution.

Example: particle_file

  # rho 2.6941 (g cm^-3)
  # E(keV)	Rayleigh(cm^2 g^-1)	Compton(cm^2 g^-1)	Photo(cm^2 g^-1)
  2.0000000e+00   1.8380000e+00   3.3750000e-02   2.2610000e+03

Example: sandpaper_file
#  x0  y0  size  
	0.027293  -0.111732  0.00446804  
	-0.118085  0.0430558  0.00524407

Input parameters

Parameters in boldface are required; the others are optional.
Name Unit Description Default
particle_file ".dat" File that contains the material cross section of the particles. NULL
sandpaper_file ".dat" File that contains spheres center positions and sizes. NULL
xwidth m X-axis width of the bounding box. 1
yheight m Y-axis height of the bounding box. 1
n_particles Max 10000000 [m] Maximum number of particles in the bounding box. 100
particle_size m Mean size of the particles. 2e-5
particle_std m Standard distribution of the particles' size. 5e-6
fixed_mu 1
slabs Default: 0 [0,1] If set 1 models the spheres as slablike particles. 0
